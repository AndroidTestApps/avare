Dieses Menü erlaubt die Erstellung und Speicherung von Flugrouten. Es können beliebig viele Waypoint hinzugefügt werden welche auch alle angezeigt werden. Die Berechnung der Distzanz, Flugdauer und BRG errechnet sich von DEP bzw. dem ersten WPT bis zur DEST bzw. dem letzten WPT.

Route anhand der in der Datenbank gespeicherten Positionen erstellen
  • Löschen sie zuerst die aktuelle Route indem sie "Lösche alle" wählen
  • Geben sie im Feld "Neue Route erstellen" jeweils DEP, die WPT und DEST durch einen Leerschlag getrennt ein, z.B. "LSZQ HOC LSZI" um von Bressaucourt über HOC nach Fricktal Schupfart zu fliegen.
  • Tippen sie nun auf den "Erstellen" Button um die Route zu laden.
  • Sämtliche Positionen welche nicht ermittelt werden können, werden verworfen.
DEP/DEST/WPT  suchen
  • Löschen sie zuerst die aktuelle Route indem sie "Lösche alle" wählen"
  • Geben sie einen WPT im Feld "Zur Route hinzufügen" ein  und wählen sie "Suchen" um eine Liste der verfügbaren WPT anzeigen zu lassen.
  • Bestätigen sie ihre Auswahl indem sie den gewünschten WPT mit dem Button rechts "Hinzufügen" zur Route hinzufügen.
  • Der neu hinzugefügte WPT ist nun in der Liste eingetragen. Wenn sie die Reihenfolge ändern möchten, dann markieren sie den entsprechenden Eintrag durch tippen und veschieben ihn mit den "Rauf/Runter" Button.
  • IMöchten sie einen Eintrag/WPT aus der Liste löschen, so markieren sie diesen durch tippen und wählen anschliessend "Löschen" um diesen aus der Liste zu lsöchen.
Route speichern
  • Möchten sie eine Route speichern, so geben sie im Feld "Route speichern" einen Namen ein und wählen anschliessend "Speichern". Bitte beachten sie dass jeweils 5 Routen angezeigt werden: wählen sie "Vorwärts" um zu überprüfen ob die Route erfolgreich gespeichert wurde.
Gespeicherte Route Laden/Löschen
  • Wählen sie "Öffnen" oder "lin umgekehrter Reihenfolge öffnen"  um eine Route zu laden.
  • Durch tippen auf den Button "Löschen" wird die Route gelöscht.
  • Um eine Route zu suchen kann im Feld "Suchen" ein Suchbegriff eingegeben werden. Alternativ kann auch durch die Liste geblätert werden indem man "vorgängig" oder "Nächste" wählt.
Gespeicherte Routen
  • Angezeigt werden jeweils 5 Routen pro Liste.
  • Mit dem Button "Erste" wird die erste Seite, mit "Letzte" die letzte Seite der Liste angezeigt.
  • Mit dem Button "Vorgängig" wird die vorhergehende, mit "Nächste" wird die nächste Seite der Liste angezeigt.
  • Mit "Filter ein - Filter aus" kann ein Suchfilter verwendet werden.
  • Mit "Liste neu laden" wird die Liste der Routen neu eingelesen und aktualisiert.
Waypoint mit dem Suchen Menü hinzufügen
  • Das Suchen Menü ist in der Lage, Orte anhand einer Adresse oder Koordinaten zu finden.
  • Tippt man länger als 2 Sekunden auf einen Eintrag im Finden Menü, erscheint unten links ein Pop-Up Menü und wählt anschliessend "+Flugroute" um den gewählten Eintrag zur Flugroute zu hinzufügen.
  • Anschliessend kann man nun weitere Positionen/WPT hinzufügen.
Flugroute mit der VFR  Karte erstellen
  • Öffnen sie das Karten Menü um eine Karte anzuzeigen.
  • Tippen sie länger als 2 Sekunden auf die gewünschte Position welche sie als WPT zur Flugroute hinzufügen möchten. Wählen sie im folgenden Pop-Up Fenster "Flugroute" um die gewählte Positione als WPT zu der Flugroute hinzufügen.
Route simulieren / auf der Karte anzeigen
  • Gehen sie zur Kartenansicht und wählen "Simulation" um den Simulationsmodus zu aktivieren.
  • Gehen sie zum Flugroute Menü und wählen unten "Aktiviert" umd die aktuell geladene Flugroute zu aktivieren.
  • Das Flugroutenmenü zeigt nun die Gesamte Strecke als auch die Distanz  zwischen den WPT an.
  • Gehen sie zurück zur Kartenansicht: die aktuelle Flugroute wird  über der Karte angezeigt.
Die Route fliegen
  • Stellen sie sicher das in der Kartenansicht der Navigationsmodus aktiviert ist und nicht der Simulationsmodus.
  • Gehen sie zum Flugroutenmenü und tippen sie auf den zweiten WPT. Info: der erste WPT ist stets DEP. Alternativ kann durch tippen auf  "Rauf-Runter" der zweite WPT markiert werden.
  • Wählen sie nun unten "Aktiviert" um die Route zu aktivieren.
  • Die verbleibende Stecke, BRG zum nächsten WPT und ETE werden in den oberen Infozeilen angezeigt.
  • Die Route (magenta) ist nun in der Kartenansicht aktiviert.
WPT während des Flug ändern mithilfe der "Kurslinie ändern" Funktion
  • Gehen sie zur Kartenansiicht während die Route aktiviert ist.
  • Tippen und ziehen sie den gewünschten WPT innerhalb von 2 Sekunden zu der neuen Positione um die Route zu ändern.
  • Der Flugroutenplan aktualisiert sich automattisch.
Reihenfolge ändern
  • Möchten sie einen WPT auslassen so tippen sie im Flugroutenmenü kurzerhand auf den nächsten WPT.
  • IInfo: nähern sie sich einem WPT so schaltet das Flugroutenmenü automatisch zum nächsten WPT um.
Flugroute deaktivieren
  • Möchten sie die aktive Flugroute deaktivieren ,so tippen sie unten im Flugroutenmü auf "Aktiviert" um vom aktiven zum inaktiven Modus zu wechseln. Die Magentafarbene Kurslinie wird nicht mehr auf der Karte angezeigt.
  • Info: wenn sie während des Flug eine neue DEST wählen so wird die aktuell aktive Flugroute automatsich deaktiviert. Möchten sie diese wieder aktivieren, tippen sie unten im Flugroutenmenü auf "Aktiviert" bzw. "Deaktiviert" um die Flugroute erneut zu aktivieren.
